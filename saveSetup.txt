1. git clone the repo

2. cd to the project folder
Install 3.11 stable version like 3.11.9 (to be compatable with d2l library) 

3. Create & activate your python virtual environment, using 3.11
py --list 
py -3.11 -m venv .venv311
.venv311\Scripts\activate

4. Install dependencies
python -m pip install --upgrade pip setuptools wheel    (Will execute on virtual venv, pip install --upgrade pip: execute on first found PATH python)
python -m pip install torch==2.0.0 torchvision==0.15.1
python -m pip install jupyter matplotlib numpy pytest
python -m pip install d2l==1.0.3

pip freeze > requirements.txt

5.Link Jupyter to venv, So Jupyter can use this environment as a kernel
python -m pip install ipykernel
python -m ipykernel install --user --name=pytorch-basics --display-name "Python (pytorch-basics)"

When run Jupyter notebooks inside VS Code, make sure kernel selector is using the .venv311(Python 3.11.0)

6. Project structure
pytorch-basics/
├── notebooks/
│   └── 01_logistic_regression.ipynb
├── src/
│   ├── models.py
│   ├── train.py
│   └── utils.py

│   └── test_models.py
├── results/
├── requirements.txt
└── README.md
